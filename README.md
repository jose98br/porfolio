# Portfolio React - Jose Manuel Benitez Ruiz

Portfolio profesional construido con React + Vite, con animaciones modernas, modo claro/oscuro y un easter egg interactivo tipo terminal.

## Stack

- React 18
- Vite 5
- Framer Motion
- CSS custom (responsive)

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Funcionalidades activas

- Diseño moderno y responsive.
- Cambio de tema claro/oscuro con transición premium y ripple desde el botón.
- Sección de proyectos, experiencia, formación y contacto.
- Easter egg `Terminal Challenge` con 3 niveles de dificultad y CTA final para compartir por LinkedIn.

## Notas

- Los datos de contenido del portfolio están centralizados en `src/data/portfolioData.js`.
- Los recursos usados por la app deben estar en `public/`.
